# MixFile JS自定义线路脚本
cookie获取教程: https://blog.csdn.net/qq_64542714/article/details/135738112 \
使用方法: 复制脚本代码后,线路选择自定义js线路，设置js线路，复制粘贴即可，cookie改为自己账号登陆后的cookie \
命令行: 复制脚本文件到运行目录，配置文件线路填写xxx.js,和文件同名即可,也可填写绝对目录 \
建议使用小号 \
也可提交您自行编写的脚本,欢迎PR
